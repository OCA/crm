==================================
Restricted Summary for Phone Calls
==================================

This module extends the functionality of CRM phone calls to support setting a
limited list of possible phone call summaries, that only the sales manager can
edit.

Configuration
=============

To configure the possible summary options, activate the developer mode and go
to:

* Go to *Sales > Configuration > Sales & Opportunities > Phone Calls >
Summaries*.

Usage
=====

* Go to *Sales > Phone Calls > Logged Calls*.
* There you can use the new *Summary* field.
